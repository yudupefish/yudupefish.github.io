java的转义字符

在控制台按tab可实现名称补全

1.\t:一个制表位，实现对齐的功能；

2.\n:换行符；

3.\:一个\

4.\“:一个”

5.\‘:一个‘

6.\r:一个回车 System.out.println（“YU学习java\r贵阳”）；

例如

//ChangeChar的使用
public class zhuanyi{
public static void main(String[] args) {
//\t:一个制表位，实现对齐的功能；
System.out.println(“猪肘子\t酱猪蹄\t猪耳朵”);
//\n:换行符；
System.out.println(“烧花鸭\n烧子鹅\n烧猪蹄”);
//\:一个
System.out.println(“水煮白菜\佛跳墙\满汉全席”);
//":一个”
System.out.println(“我想吃："雪媚娘，抹茶蛋糕，珍珠奶茶"“);
//':一个‘
System.out.println(“我想吃：'雪媚娘，抹茶蛋糕，珍珠奶茶'“);
//\r:一个回车 System.out.println（“YU学习java\r贵阳”）
//\r代表回到开头继续替换输出如
System.out.println(“YU学习java\r贵阳”);
//想要另外起一行则需要用\r\n如
System.out.println(“YU学习java\r\n贵阳”);
}
}

初学java容易犯的错误

1.找不到文件

解决方法：源文件名不存在或者写错，或者当前路径错误

2.主类名的文件名不一致

解决方法：声明为public的主类应与文件名一致，否则编译失败

3.缺少分号

解决方法：编译失败，注意错误出现的行数，再到源代码中指定位置改错

4.常见错误总结

学习编程最容易犯的错是语法错误（初学者）。Java要求你必须按照语法规则编写代码。如果你的程序违反了语法规则，例如：忘记了分号、大括号、引号，或者拼错了单词，java编译器都会报语法错误。尝试着去看懂编译器会报告的错误信息。

注释comment

java中的注释类型

1.单行注释：//注释文字

2多行注释：/*注释文字 */

3文档注释：注释内容可以被JDK提供的工具javadoc所解析，生成一套以网页文件形式体现的该程序的说明文档，一般写在类

文档注释使用方法

javadoc标签

javadoc注释规范

使用细节

1.被注释文字，不会被JVM解释执行

2.多行注释里面不允许有多行注释嵌套

Java代码规范

1.类，方法的注释，要以javadoc的方法来写。

2.非Javadoc的注释，往往是给代码的维护者看的，着重告诉读者为什么这样写，如何修改，注意什么问题等。

3.使用tab操作，实现缩进，默认整体向右边移动，使用时按住shift+tab时整体向左移动。

4.运算符和=两边习惯性各加一个空格。比如2 + 4 * 5 + 345 - 89

5.源文件使用utf-8编码

6.行宽度不超过80字符

7.代码编写次行风格和行尾风格（！！）

DOS命令

.DOS介绍

DOS：Disk Operating System 磁盘操作系统，简单说一下windows的目录结构。

相关知识补充：

相对路径：从当前目录开始定位，形成的一个路径，即相对路径

即：从子目录访问别的子目录需要返回（..\）到根目录在从新访问到别的子目录

绝对路径：从根目录开始定位形成的路径，称之为绝对即：直接从根目录访问到别的子目录

常用的dos命令

1.查看当前目录是有什么

dir dir d:\abc2\test200

2.切换到其它盘下：盘符号 cd

如：切换到 c盘 cd/D c:

3.切换到当前盘的其他目录下 ..上一级目录

如：cd d:\abc2\test200 cd .. \ ..\abc2\test200

4.切换到上一级：

如： cd ..

5.切换到根目录：cd\

如： cd\

6.增加文件夹：md

7.删除文件夹： rd

8.查看盘下所有内容：tree

9.清屏：cls

10.退出：exit
